## ✅ 系统功能

### 1. 80 AI玩家初始化
- **自动生成** 80个中文名AI玩家
- **随机分配** 初始资源（金币/钻石/VIP等级）
- **行为模式** 分类：普通型(40)、激进型(25)、保守型(15)

### 2. 道具分配系统

**可分配道具类型:**
| 类型 | 数量 | 示例 |
|-----|------|------|
| 扑克牌皮肤 | 5种 | 粉色萌猫、水晶魔法、东方神龙等 |
| 头像框 | 3种 | 黄金框、钻石框、传说框 |
| 称号 | 3种 | 战士、大师、传说 |
| 嘲讽包 | 1种 | 嘲讽表情包 |
| 经验卡 | 2种 | 2倍、3倍经验卡 |
| 幸运符 | 1种 | 幸运加成 |

### 3. 消费模拟
- 每个AI随机购买 **3-8个道具**
- 自动扣除相应货币
- 记录消费统计

### 4. 自动装备
- AI自动装备已购买的皮肤/头像框/称号
- 根据装备显示不同视觉效果
