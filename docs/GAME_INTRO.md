# 命运塔·首登者 - 游戏介绍

**英文名称:** Tower of Fate: First Reacher  
**游戏类型:** 多人实时卡牌对战  
**核心玩法:** 爬塔竞技 + 团队协作 + 排位晋升

---

## 🎮 游戏简介

《命运塔·首登者》是一款创新的多人实时卡牌对战游戏。玩家将扮演命运挑战者，通过出牌、策略和运气的较量，攀塔晋级，争夺"首登者"的至高荣誉。

### 游戏特色

- ⚡ **实时对战** - 4人实时匹配，快节奏对局
- 🎴 **天命牌系统** - 特殊技能逆转战局
- 🏆 **首位荣耀** - 登顶成为守卫，守住13轮获胜
- 👥 **团队竞技** - 2v2/3v3/4v4/5v5团队对战
- 🌍 **全球锦标** - 40+城市巡回赛
- 🎯 **连胜挑战** - 连续13次登顶挑战

---

## 🎯 核心玩法

### 基础规则

1. **爬塔晋级** - 从2层开始，通过出牌匹配守卫牌晋级
2. **匹配条件** - 花色相同 或 数字相同 即可晋级
3. **完美匹配** - 花色和数字都相同，获得天命牌
4. **登顶获胜** - 到达A层(13层)即可获胜

### 守卫模式（创新玩法）

- 登顶后变成**守卫**
- 守卫出守卫牌，守住塔顶
- 每守住一轮 +50分
- 守住**13轮** = 胜利
- 其他玩家可用"⬇️下降"天命牌攻击守卫
- 守卫被淘汰后从2层重新开始

---

## 🎴 天命牌系统

开局即给 **3张** 天命牌，随时可用：

| 天命牌 | 效果 |
|-------|------|
| ⚡ 全军突击 | 全队晋级 |
| 🔄 换牌 | 重新抽手牌 |
| 👁️ 看牌 | 查看守卫牌 |
| 🚀 带人 | 带队友晋级 |
| ❌ 踢人 | 让对手降一层 |
| ⬇️ 下降 | **淘汰守卫！** |

**额外获取：**
- 登顶后队友每上一层，获得1张天命牌
- 完美匹配获得额外天命牌

## 🏆 游戏模式

### 1. 经典对战 (1vs3AI)
单人挑战3个AI，练习技巧

### 2. 排位赛
- 8大段位：青铜→白银→黄金→铂金→钻石→星耀→王者→首位好运
- 每段位13级，连续13次通关晋级
- 赛季制，30天一个赛季

### 3. 团队赛
| 模式 | 团队人数 | 共享手牌 | 每人选牌 |
|-----|---------|---------|---------|
| 2v2 | 2人 | **10张** | 1张 |
| 3v3 | 3人 | **15张** | 1张 |
| 4v4 | 4人 | **20张** | 1张 |
| 5v5 | 5人 | **50张** | 1张 |

### 4. 锦标赛
- 全球40+城市巡回赛
- 积分报名，高额奖金
- 20分钟倒计时，紧张刺激

### 5. 连胜模式
- 连续13次登顶挑战
- 5档连胜奖励
- 难度递增机制

---

## 💎 积分系统

### 获取积分
- 胜利：+100分
- 花色/数字匹配：+20分
- 完美匹配：+100分 + 天命牌
- 守卫每轮：+50分
- 淘汰守卫：+200分

### 使用积分
- 锦标赛报名
- 商城购买
- 解锁特殊功能

---

## 🛡️ 守卫模式详解

### 成为守卫
- 个人/团队登顶后变成守卫
- 改变外观，显示守卫标识
- 每轮+50分奖励

### 守卫的挑战
- 需守住13轮攻击
- 其他玩家会全力进攻
- "下降"天命牌是最大威胁

### 守卫的策略
- 合理使用天命牌
- 与队友配合（团队赛）
- 预判对手出牌
