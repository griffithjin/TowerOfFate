## 📐 塔模型标准

### 视觉呈现（正立塔）

```
        ▲ 塔尖
        │
   ┌────┴────┐
   │   A层   │  ← 第13层（最顶层）
   │   K层   │  ← 第12层
   │   Q层   │  ← 第11层
   │   J层   │  ← 第10层
   │   🔟层  │  ← 第9层
   │   9️⃣层  │  ← 第8层
   │   8️⃣层  │  ← 第7层
   │   7️⃣层  │  ← 第6层
   │   6️⃣层  │  ← 第5层
   │   5️⃣层  │  ← 第4层
   │   4️⃣层  │  ← 第3层
   │   3️⃣层  │  ← 第2层
   │   2️⃣层  │  ← 第1层（最底层/起始层）
   └────┬────┘
        │
        ▼ 塔底座
```

### 关键规则

| 属性 | 值 | 说明 |
|-----|-----|------|
| 最底层 | 第2️⃣层 | 玩家起始位置，塔底座 |
| 最顶层 | 第A层 | 终极目标，塔尖 |
| 层数顺序 | 2️⃣→A | 从下到上递增 |
| 渲染方向 | 从下到上 | flex-direction: column |
| 总层数 | 13层 | 对应扑克牌2-A |

### CSS实现

```css
/* 塔容器 - 正立显示 */
.tower-container-unified {
    display: flex;
    flex-direction: column; /* 从上到下渲染 */
    width: 100%;
    height: 100%;
}

/* 塔层 */
.tower-level-unified {
    flex: 1;
    /* 第1层(2️⃣)在底部，第13层(A)在顶部 */
}

/* 第2️⃣层（最底层） */
.tower-level-unified[data-level="1"] {
    border-bottom-left-radius: 8px;
    border-bottom-right-radius: 8px;
}

/* A层（最顶层） */
.tower-level-unified[data-level="13"] {
    border-top-left-radius: 8px;
    border-top-right-radius: 8px;
}
```

### JavaScript渲染

```javascript

// 渲染塔（从下到上）

    const levelName = LEVEL_NAMES[i - 1];
    // i=1 → 2️⃣层（最底部）

```

## 👤 玩家icon系统

### 起始位置
- 所有玩家从**第2️⃣层**（最底部）开始
- 玩家icon显示在对应层数的右侧

### 攀登方向
- 向上攀登：第2️⃣层 → 第3️⃣层 → ... → 第A层
- 每次成功晋级，层数+1

### 多玩家显示
- 同一层可以有多个玩家icon
- icon水平排列，自动偏移
