## ✨ 霓虹灯效果（灯红酒绿）

### 实现代码
```css
@keyframes neonPulse {
    0%, 100% { 
        box-shadow: 0 0 10px currentColor, 0 0 20px currentColor;
        filter: brightness(1);
    }
    50% { 
        box-shadow: 0 0 20px currentColor, 0 0 40px currentColor, 0 0 60px currentColor;
        filter: brightness(1.3);
    }
}

.tower-level.neon {
    animation: neonPulse 1.5s ease-in-out infinite;
}
```

### 效果塔
- 东方明珠 (粉色霓虹)
- 哈利法塔 (金色霓虹)
- 广州塔 (青色霓虹)
