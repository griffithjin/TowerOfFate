## 🚀 快速启动

### 方式1: 使用启动脚本 (推荐)

```bash
# 进入项目目录
cd ~/Desktop/TowerOfFate

# 启动所有服务
./start.sh

# 停止所有服务
./stop.sh
```

### 方式2: 手动启动

```bash
# 1. 启动游戏服务器
cd ~/Desktop/TowerOfFate/server
python3 game_server.py

# 2. 启动Web客户端 (新终端)
cd ~/Desktop/TowerOfFate/web_client
python3 -m http.server 8080

# 3. 启动后台管理 (新终端)
cd ~/Desktop/TowerOfFate/admin
python3 -m http.server 8081
```

## 🛠️ 开发指南

### 添加新功能

1. **后端开发:**
   - 在 `server/` 目录创建新模块
   - 在 `game_server.py` 添加消息处理器
   - 更新 WebSocket 路由

2. **前端开发:**
   - 在 `web_client/` 创建新页面
   - 在 `new_index.html` 添加菜单入口
   - 添加对应的功能卡片到 `features.html`

3. **后台管理:**
   - 在 `admin/index.html` 添加管理功能
   - 添加对应的数据展示和操作按钮

### 数据库修改

当前使用内存存储，数据在重启后丢失。
如需持久化，修改以下文件:
- `server/player_data.py` - 玩家数据
- `server/stats.py` - 统计数据

## 🔧 故障排除

### 端口被占用
```bash
# 查找占用端口的进程
lsof -i :7778
lsof -i :8080
lsof -i :8081

# 结束进程
kill -9 <PID>
```

### 依赖安装
```bash
# 安装Python依赖
pip3 install websockets
```

### 浏览器缓存
- 按 `Ctrl+Shift+R` 强制刷新
- 或清除浏览器缓存
