## 🎮 核心逻辑清单

### 1. 个人赛 (solo_game.html)

#### 田字四象限布局 ✅
```
┌───────────────┬───────────────┐
│  守卫牌(上)    │               │
│  天命牌(下)    │   核心塔      │
│  (日字形)      │  (13层世界名塔)│
├───────────────┼───────────────┤
│   手牌横条     │   对局记录    │
│  (可滚动5-18张) │   (实时嘲讽)  │
└───────────────┴───────────────┘
```

#### 塔层命名体系 ✅
| 层级 | 显示名称 |
|-----|---------|
| 1-9层 | 第2️⃣层-第🔟层 |
| 10层 | 第J层 |
| 11层 | 第Q层 |
| 12层 | 第K层 |
| 13层 | 第A层(顶层) |

#### 手牌系统 ✅
- 初始5张牌
- 每次晋级+1张牌
- 最多18张牌
- **自动去重**（同花色同数字不重复）
- 横向滚动展示

#### 世界名塔系统 ✅
| 塔名 | 国家 | 形状特征 |
|-----|------|---------|
| 埃菲尔铁塔 | 法国 | 锥形（下宽上窄） |
| 东京塔 | 日本 | 红色锥形 |
| 帝国大厦 | 美国 | 方正摩天楼 |
| 双子塔 | 马来西亚 | 双塔造型 |
| 哈利法塔 | 阿联酋 | 尖顶摩天楼 |
| 东方明珠 | 中国 | 球体串联 |
| 比萨斜塔 | 意大利 | 倾斜圆柱 |
| 伦敦塔桥 | 英国 | 双塔桥梁 |

#### 守卫模式规则 ✅
```
登顶条件: 到达第A层(13层)
守卫任务: 成功防御13次攻击
关键规则: 累计13次即可(不需要连续)
获胜条件: AI 13次无法匹配守卫牌
```

#### 11项天命牌系统 ✅

**增益牌(对自己/队友)**
| 牌名 | 图标 | 效果 |
|-----|------|------|
| 突击 | ⚡ | 立即+1层 |
| 换牌 | 🔄 | 重新抽牌 |
| 透视 | 👁️ | 看守卫牌 |
| 守卫 | 🛡️ | 本轮必胜 |
| 双倍 | ✨ | 下次+2层 |
| 护盾 | 🔒 | 免疫一次 |

**损益牌(对对手)**
| 牌名 | 图标 | 效果 |
|-----|------|------|
| 踢人 | 👢 | 对手-1层 |
| 下降 | ⬇️ | 守卫回2层 |
| 冰冻 | ❄️ | 对手停一轮 |
| 偷看 | 👀 | 看对手牌 |
| 复制 | 📋 | 复制上轮牌 |

#### AI完整智能 ✅
- AI知道游戏规则
- AI会自动尝试晋级(35%成功率)
- **AI登顶后也会变成守卫**
- AI会嘲讽玩家
- AI会完成完整的守卫模式

#### 嘲讽语音系统 ✅
- **100条嘲讽语**分类存储
  - 自信类(20条)
  - 读心类(15条)
  - 运气相关(20条)
  - 夸奖对手(15条)
  - 挑衅类(15条)
  - 幽默类(15条)
- 语音自动播放(TTS)
- 气泡动画展示
